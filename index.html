Welcome to EOS Factory

EOS Factory is a smart-contract development framework, made by Tokenikafor compiling, testing and deploying EOS smart-contracts.

It's going to have a similar functionality to Ethereum's Truffle Framework.

It will be composed of two layers:

  • C++ interface for interacting with an EOS node (similar to block.one's cleos)
  • Python interface which is a convenient human-oriented wrapper around the above C++ interface

Using Python will allow us to build interactive tutorials for EOS smart-contracts.

EOS Factory is still under development. We plan to release it before June 2018.
