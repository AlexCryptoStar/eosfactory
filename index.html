Welcome to EOS Factory

EOS Factory is a framework for compiling, testing and deploying EOS smart-contracts, similar to Ethereum's Truffle Framework.

It's composed of two layers:

  • C++ interface for interacting with an EOS node (similar to block.one's cleos)
  • Python interface which is a convenient human-oriented wrapper around the above C++ interface

Using Python will allow us to build interactive tutorials for EOS smart-contracts
